6  Tarea 3 - Ley de enfriamiento

La ley de enfriamiento de Newton nos dice que la tasa de pérdida de calor de un cuerpo es directamente proporcional a la diferencia de temperatura entre el cuerpo y su entorno.

De forma simplificada \[ \frac{d}{dt} T = -\kappa \big[ T(t)-T_a \big] \]

Al resolver la ecuación anterior obtenemos \[ \ln \left| \frac{T(t)-T_a}{T(t_0)-T_a} \right| = -\kappa (t-t_0) \]

Code 
Sxy = np.sum(x*y)
Sx  = np.sum(x)
Sy  = np.sum(y)
Sxx = np.sum(x*x)
Fac = n*Sxx-Sx*Sx
m   = (n*Sxy-Sx*Sy)/Fac
b   = (Sxx*Sy-Sx*Sxy)/Fac
print ('m = {0:.2E} 1/min'.format(m))
print ('b = {0:.2E} '.format(b))
